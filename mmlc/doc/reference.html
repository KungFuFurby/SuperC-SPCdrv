SuperC mmlconv command reference

Note: This mml distinguishes between upper - and lowercase letters

Command Description Implements
/* ... */ Multiline comment ○
// Single line comment ○
#title ... Set title of music ×
#game ... Set game title ×
#composer ... Set music composer name ×
#dumper ... Set SPC Dumper info ×
#comment ... Set SPC Comment ×
#track ... Set track (1 ~ 10) ○
#timebase ... Set resolution of SPC
default is 48(0x30)		 ○
#swap<> Swap octave up/down ○
#tone ... Define tones for music.

Syntax:

ADSR MODE:

#tone n1,n2,n3,n4,n5,n6,n7,n8,n9

n1 ... BRR - BRR name or BRR number (e.g. "smpl/piano.brr" or 0)

n2 ... Pitch multiple(0~255)

n3 ... Tuning(-128~127)

n4 ... 1(ADSR)

n5 ... Attack Rate(0~15)

n6 ... Decay Rate(0~7)

n7 ... Sustain Level(0~7)

n8 ... Sustain Rate(0~31)

n9 ... Release Rate(0~31)

GAIN MODE:

#tone n1,n2,n3,n4,n5,n6,n7,n8

n1 ... BRR - BRR name or BRR number (e.g. "smpl/piano.brr" or 0)

n2 ... Pitch multiple(0~255)

n3 ... Tuning(-128~127)

n4 ... 0(GAIN)

n5 ... Atack Gain mode 1 (0: Direct / 1: Linear Decrease / 2: Exp Decrease / 3: Linear Increase / 4: Bent Increase)

n6 ... Atack Gain param 1 (Direct mode: 0~127 / Other modes: 0-31)

n7 ... Release Gain mode 2

n8 ... Release Gain param 2

e.g. #tone "smpl/lead.brr",5,0,1,15,7,7,0,18

 ○
#drum ... Define drum set for music.

Syntax:

#tone command syntax + panpot(0-63) + note number(0~83)

e.g. #drum "smpl/kick.brr",4,0,1,15,3,4,29,31,32,48

 ○
#macro MACRONAME ... Define replace macro
ex. #macro MACNAME @2		 ○
$(MACRONAME) Expand the specified macro ○
(0)xnn Specify hex value
ex.0xef, x3f		 ○
o Specify octave ○
< Octave up/down
Default is down		 ○
> Octave up/down
Default is up		 ○
c,d,e,f,g,a,b Note
You can also use '+'(sharp) and '-'(flat)		 ○
r Rest ○
^ Tie ○
/ Specify music loop point
Default, music isn't loop		 ○
qn1 Set gatetime ratio
You can specify 1 ~ 8
Means: 1 = 12.5%, 2 = 25%, 3 = 37.5% ... 8 = 100%
If note is long(more than 96ticks), ratio becoms sloppy.		 ○
kn1 Set velocity
You can specify 1 ~ 16
Means: 1 = 6.25%, 2 = 12.5%, 3 = 18.75% ... 16 = 100%		 ○
@n1 Set instrument from music tone table ○
@@n1 Set instrument from internal tone table ○
pn1(,n2) Set panpot △
vn1(,n2) Set channel volume △
Vn1(,n2) Set master volume △
mn1,n2(,n3) Set moduration ×
tn1(,n2) Set Tempo
Specify the value of bpm.		 △
Xn1 Sequence command direct insert ○
%d(n1) Enable / Disable drum set
%d ... enable / %d0 ... disable		 ○
%e(n1) Enable / Disable ch-echo
%e ... enable / %e0 ... disable		 ○
%p(n1) Enable / Disable portament
%p ... enable / %p0 ... disable		 ○
%rn1,n2 Enable / Disable reverse surround
n1 ... Left / %n2 ... Right // 1=on, 0=off		 ○
